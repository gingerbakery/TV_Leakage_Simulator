# Ray path 유형별 표시 필터

## 목적
- Ray tracing 결과에서 여러 경로가 동시에 겹쳐 보이는 문제를 줄인다.
- 설계 판단에 가장 중요한 Receiver 도달 경로를 우선 확인할 수 있게 한다.

## 변경 내용
- Web UI 버전을 `v0.9.14`로 변경했다.
- 왼쪽 `Result` 메뉴에 `3D Ray path 표시` 체크박스 영역을 추가했다.
- 저장된 ray path를 다음 표시 유형으로 분리했다.
  - Receiver 도달: 밝은 연두색, 기본 표시
  - Direct / 최초 진행: 파란색, 기본 표시
  - Specular: 주황색, 기본 표시
  - Lambertian: 보라색, 기본 표시
  - Gaussian: 청록색, 기본 표시
- `Receiver only`, `All on`, `All off` 빠른 설정을 추가했다.
- Receiver에 도달한 다회 반사 경로는 전체 구간을 하나의 밝은 연두색 경로로 표시한다.
- 표시 개수는 `현재 표시 경로/전체 저장 경로` 형식으로 보여준다.

## 동작 원칙
- Receiver 도달 여부가 반사 모델보다 높은 시각화 우선순위를 가진다.
- Direct 및 반사 모델 필터는 Receiver에 도달하지 못한 저장 경로에 적용한다.
- 필터 변경은 ray tracing을 다시 계산하지 않고 3D overlay만 즉시 다시 그린다.

## 검증 항목
- 기본 상태에서 모든 저장 경로 유형이 표시되는지 확인한다.
- 각 체크박스와 빠른 설정 버튼이 즉시 overlay에 반영되는지 확인한다.
- ray tracing 결과가 무효화되거나 실패하면 표시 개수가 초기화되는지 확인한다.
