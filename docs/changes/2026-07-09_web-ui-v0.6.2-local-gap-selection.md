# Web UI v0.6.2 ROI 독립 local gap selection

## 목적
- ROI를 선택하지 않았더라도 gap 대상 설정이 가능하도록 ROI와 gap 선택 흐름을 분리한다.
- `선택 면만 이동` 모드에서 single face 클릭만 가능한 제약을 줄이고, surface cluster / 다중 선택 / 영역 선택까지 지원한다.

## 반영 내용
- Web UI 버전을 `v0.6.2`로 갱신
- `local face move` 대상이 ROI 선택에 의존하지 않도록 수정
  - ROI 미선택이어도 전체 모델 기준으로 local gap target 선택 가능
- local face 선택 UX 추가
  - `Click`: surface cluster 선택
  - `Ctrl+Click`: cluster 추가 또는 해제
  - `Shift+Drag`: 사각형 영역 선택
- cluster 선택을 위해 viewer에서 mesh face adjacency를 구성하고, 인접 + 유사 normal 기준으로 face 군집을 확장
- Run OFF 상태가 CAD 로드 후에도 유지되도록 보정

## 기대 효과
- ROI를 일부러 안 쓰는 경우에도 gap 검토를 바로 진행 가능
- local face move가 실제 기구 검토 흐름에 더 가까운 방식으로 동작
- 면 단위 수작업 클릭 부담 감소

## 검증
- `py_compile run_web.py` 통과
- HTML 템플릿 생성 확인
