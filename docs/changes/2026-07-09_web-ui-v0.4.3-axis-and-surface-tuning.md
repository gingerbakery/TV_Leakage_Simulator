# Web UI v0.4.3 axis zoom 연동 / surface 가시성 개선

## 목적
- axis triad가 줌 변화에 비해 충분히 작아지지 않는 문제를 보정한다.
- `Surface`, `Surface + Edge` 모드의 경계/명암 대비를 높여 형상 시인성을 개선한다.

## 반영 내용
- world axis overlay를 projected endpoint 직접 사용 방식에서 pixel-length 제어 방식으로 조정
- mini axis triad도 projected delta 직접 사용 대신 정규화 벡터 + pixel-length 제어 방식으로 조정
- axis 길이는 아래 두 요소를 함께 반영
  - 줌(`state.transform.distance`)
  - 사용자 슬라이더(`Axis size`)
- `Surface` 모드에 subtle edge stroke 추가
- `Surface + Edge` 모드의 경계선 대비 강화
- shaded 색상 톤을 조금 더 밝고 구분감 있게 조정

## 기대 효과
- 확대 시 axis triad가 상대적으로 더 자연스럽게 작아 보임
- complex STEP 형상에서도 면/경계 인지가 더 쉬워짐
