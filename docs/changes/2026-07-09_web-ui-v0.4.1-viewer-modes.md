# Web UI v0.4.1 viewer render mode / 좌표축 표시

## 목적
- 복잡한 CAD에서 기존 반투명 wireframe만으로는 형상 시인성이 떨어지는 문제를 개선한다.
- 3D viewer에서 원점과 XYZ 방향을 인지하기 어려운 문제를 개선한다.

## 반영 내용
- Render mode 추가
  - `Wireframe`
  - `Surface`
  - `Surface + Edge`
- `Surface` 계열 모드에서는 면이 채워져 보이도록 간단한 shaded 렌더 적용
- 좌표축 표시 추가
  - world origin 기준 `X/Y/Z` 축 오버레이
  - 좌하단 고정 mini axis triad
- 좌표 정보 패널 추가
  - `Origin`
  - `Center`
  - `BBox min`
  - `BBox max`

## 구현 원칙
- 복잡한 3D 엔진 도입 없이 현재 canvas renderer 위에서 경량 구현
- ROI 선택 로직은 유지하고, 렌더 표현만 개선

## 기대 효과
- 기어처럼 edge가 많은 STEP 모델도 면 중심으로 더 쉽게 형상 파악 가능
- 사용자가 회전/줌 중에도 XYZ 방향과 원점 기준을 잃지 않음
