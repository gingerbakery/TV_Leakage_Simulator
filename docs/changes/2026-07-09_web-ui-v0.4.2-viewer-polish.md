# Web UI v0.4.2 viewer mode 가시성 / axis size 가변화

## 목적
- 렌더 모드가 실제로 바뀌었는지 한눈에 확인하기 어려운 문제를 개선한다.
- 모델 확대/축소 시 좌표축이 상대적으로 너무 크게 보이는 문제를 개선한다.

## 반영 내용
- 기존 드롭다운형 render mode를 버튼형 segmented control로 변경
  - `Wireframe`
  - `Surface`
  - `Surface + Edge`
- 현재 선택된 렌더 모드를 우측 badge로 항상 표시
- `Axis size` 슬라이더 추가
  - `50% ~ 150%`
- mini axis triad 크기가 줌(`distance`)에 따라 자동으로 변하도록 조정
- axis 길이 계산에도 사용자 지정 비율(`Axis size`) 반영

## 기대 효과
- 렌더 모드 전환 여부를 즉시 인지 가능
- 확대 시 좌표축이 과도하게 커 보이지 않고, 축소 시 너무 작아지지도 않음
