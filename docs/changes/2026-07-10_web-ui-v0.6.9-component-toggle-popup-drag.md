# Web UI v0.6.9 - Component Toggle and Draggable Transform Popup

## Why
- `선택 해제` 버튼 없이도 component tree에서 클릭만으로 선택/해제를 반복하는 쪽이 더 CAD 프로그램에 가깝다.
- Transform popup이 우측에 고정되어 있으면 모델을 가릴 수 있어, 사용자가 직접 위치를 옮길 수 있어야 했다.

## What changed
- `Components` 탭의 `선택 해제` 버튼 제거
- component tree row 클릭 동작 변경
  - 1회 클릭: 선택
  - 동일 component 재클릭: 선택 해제
- 오른쪽 `Transform popup`에 드래그 이동 기능 추가
  - popup 헤더를 마우스로 끌어서 3D viewer 안 원하는 위치로 이동 가능
  - 이동 후 다시 열어도 마지막 위치 유지

## Implementation notes
- `run_web.py`
  - tree selection은 `setSelectedGapObject(..., additive=true)` 기반 토글로 통일
  - popup drag 상태는 `state.popupPosition`, `state.popupDrag`로 관리
  - viewer 영역 밖으로 나가지 않도록 위치 clamp 적용
