# Web UI v0.6.3 gap drag-box selection / popup reposition

## 목적
- 3D viewer에서 gap 대상(surface/component)을 드래그 박스로 선택할 수 있도록 추가한다.
- viewer 위 transform 입력창이 모델 중앙을 가리는 문제를 줄이기 위해 우측 도킹 위치로 정리한다.

## 반영 내용
- Web UI 버전을 `v0.6.3`으로 갱신
- Gap Step에 `선택 방식` 추가
  - `클릭 선택`
  - `드래그 박스 선택`
- `부품 전체 이동` 모드
  - 드래그 박스 범위 내 component를 gap target으로 선택
  - `Ctrl+Drag`로 add/remove 가능
  - 복수 component 선택 시 공통 transform 적용
- `선택 면만 이동` 모드
  - 드래그 박스 범위 내 face cluster를 local target으로 선택
  - 기존 click cluster 선택도 유지
- floating `Transform input` 팝업은 클릭 위치 대신 viewer 우측에 도킹되도록 위치 로직 변경

## 기대 효과
- CAD 툴처럼 viewer에서 바로 대상 범위를 끌어서 선택 가능
- component와 surface 모두에 대해 drag selection workflow 제공
- 입력창이 triad/모델 중심을 덜 가려 시인성 향상

## 검증
- `py_compile run_web.py` 통과
- HTML 템플릿 생성 확인
