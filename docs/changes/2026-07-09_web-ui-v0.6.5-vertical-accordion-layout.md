# Web UI v0.6.5 - Vertical Accordion Layout

## Why
- 현재 좌측 `수평 탭`은 CAD 프로그램처럼 순차 작업하기에 직관성이 떨어졌다.
- 사용자 요청에 따라 예전처럼 `세로형` 흐름을 기본으로 되돌리되, 비교용 `가로형`도 남겨야 했다.

## What changed
- 좌측 메뉴 기본 레이아웃을 `세로 아코디언`으로 변경
- 상단에 `세로형 / 가로형` 전환 토글 추가
- 공통 메뉴 순서를 아래로 통일
  - `CAD`
  - `Components`
  - `ROI 설정`
  - `Transform`
  - `Material`
  - `Ray tracing`
  - `Result`
- `Material`, `Ray tracing`, `Result`는 이후 기능 확장을 고려한 독립 섹션으로 분리
- 결과 메시지 표시 위치를 `Result` 섹션 내부로 이동

## Implementation notes
- `run_web.py`
  - `sidebarNavShell[data-layout]` 기반으로 세로/가로 레이아웃 전환
  - `switchSideTab()`은 가로 탭 버튼과 세로 아코디언 헤더를 동일 계약으로 제어
  - `setSidebarLayout()` 추가로 기본값을 `vertical`로 고정

## Expected UX
- 기본 상태에서 좌측 메뉴가 위에서 아래로 펼쳐지는 방식으로 보인다.
- 필요하면 `가로형` 버튼으로 즉시 이전 스타일을 비교할 수 있다.
- 향후 `Material ~ Result` 기능이 커져도 메뉴 구조를 다시 바꾸지 않고 확장 가능하다.
