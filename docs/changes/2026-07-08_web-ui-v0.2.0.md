## 2026-07-08: Web UI v0.2.0

### 반영 항목
- `run_web.py`
  - Web UI 버전을 `v0.2.0`으로 상향
  - 출력 파일 링크 안정화:
    - 실행 결과 파일을 서버 메모리 맵(`OUTPUT_FILE_INDEX`)에 등록해 `include custom output_dir`
    - `/outputs/<file>` 요청 시 등록된 파일 경로에서 우선 응답
  - 기본 선택 소재를 `black_pc_resin`으로 명시
  - `material_override` 값 타입 정합성 문제 정리 및 문자열 이스케이프 정리

- `src/leakage_simulator/engine.py`
  - 소재 오버라이드 경로 정비:
    - `material_override`(reflectance/roughness/ratio)를 받아 기본 소재 기반 custom 프로필 생성
    - `material_preset`가 없을 때도 `black_pc_resin` 기반 override 적용
  - 잘못된 `materials_override` 입력경로 제거로 런타임 충돌 가능성 차단

- `run_gui.py`
  - GUI 파싱 오류(이스케이프된 문자열) 수정

### 테스트
- `python -m py_compile` 로 변경 파일 컴파일 통과
- `run.py` CLI 실행으로 JSON/CSV/PNG/HTML 산출 확인
- `run_web.py` 서버 실행 후 `/health` 응답이 `ok web_ui_version=0.2.0` 확인
- Python in-process POST 시뮬레이션으로 Web UI 응답에 `Run complete` 및 출력 링크 생성 확인
