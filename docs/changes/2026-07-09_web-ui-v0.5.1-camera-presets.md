# Web UI v0.5.1 camera preset 추가

## 목적
- CAD viewer에서 특정 평면과 역방향 평면을 빠르게 확인할 수 있도록 카메라 프리셋을 추가한다.

## 반영 내용
- viewer 상단에 camera preset 버튼 추가
  - `Fit`
  - `Iso`
  - `XY`
  - `-XY`
  - `XZ`
  - `-XZ`
  - `YZ`
  - `-YZ`
- plane view를 위해 pitch clamp 범위를 `±1.55`까지 확장

## 의도
- `XY`, `XZ`, `YZ`는 각 평면을 정면으로 보기 위한 뷰
- `-XY`, `-XZ`, `-YZ`는 반대 방향에서 같은 평면을 확인하기 위한 뷰
- `Fit`은 현재 뷰 방향을 유지하면서 distance를 기본값으로 되돌림
