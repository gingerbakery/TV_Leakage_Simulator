# Web UI v0.3.5 ROI label polish

## Summary
- Renamed ROI selection labels to focus on user selection methods instead of implementation terms.
- Removed wording like `left panel` from the ROI selection bar and guidance text.

## Updated labels
- `ROI selection mode` -> `ROI 선택 방식`
- `Combined (3D + left panel)` -> `복합 선택`
- `3D click only` -> `3D view에서 선택`
- `Left panel only` -> `Component 선택`
- `Clear ROI` -> `ROI 초기화`

## Supporting text
- Updated helper text and viewer tip messages to match the new naming style.
