# Web UI v0.6.6 - Accordion Multi-Open and Model Import Label

## Why
- 첫 번째 메뉴는 실제 기능이 `CAD 전체`가 아니라 사실상 `model import` 한 가지라서 더 직관적인 이름이 필요했다.
- 세로형 메뉴는 CAD 프로그램처럼 여러 작업 패널을 동시에 펼쳐두는 사용성이 더 중요했다.

## What changed
- 첫 번째 메뉴 라벨을 `CAD`에서 `Model import`로 변경
- 세로형 아코디언에서
  - 한 번 클릭하면 열림
  - 한 번 더 클릭하면 닫힘
  - 여러 메뉴를 동시에 열어둘 수 있음
- 드롭다운 열림/닫힘 애니메이션을 조금 더 부드럽게 조정

## Implementation notes
- `run_web.py`
  - `state.openSidePanels`로 세로형 열린 패널 집합 관리
  - `syncSidePanels()`로 세로형/가로형 상태 반영 로직 분리
  - `switchSideTab(tab, { forceOpen })` 형태로 내부 제어 일관화

## UX intent
- 사용자는 `Model import`, `Components`, `Transform` 등을 동시에 펼쳐두고 작업할 수 있다.
- 내부 자동 전환이 필요할 때도 이미 열린 섹션이 불필요하게 닫히지 않는다.
