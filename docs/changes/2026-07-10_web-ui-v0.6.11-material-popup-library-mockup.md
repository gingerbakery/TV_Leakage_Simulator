# Web UI v0.6.11 - Material popup / library mockup

## Summary
- `Material` 지정 동작을 `Transform`과 유사하게 오른쪽 `3D viewer popup`으로 옮겼다.
- 왼쪽 `Material library` 탭은 `재질/표면 특성 목록`, `custom surface 등록`, `BSDF asset 등록`, `assignment 목록` 중심으로 재구성했다.
- `Components` 탭의 `Material` 버튼을 누르면 대상 부품이 하이라이트되고 Material popup이 열린다.

## Updated files
- `run_web.py`

## Key UI changes
- 오른쪽 viewer에 `Material assign` 팝업 추가
  - `Apply mode`
  - `Base material`
  - `Surface finish`
  - `Saved optical profile`
  - `Apply / To faces / Save profile`
- 왼쪽 `Material library` 패널 개편
  - `Base materials`
  - `Surface finishes`
  - `Custom surface register`
  - `BSDF assets`
  - `Saved optical profiles`
  - `Assignments`

## Interaction changes
- `Components -> Material` 클릭 시
  - 부품 하이라이트
  - Material popup 오픈
  - Material library 탭 동기화
- popup에서 `Apply`
  - part assignment 또는 face override를 mockup 상태로 프로젝트 assignment 목록에 반영
- popup에서 `Save profile`
  - 현재 draft를 optical profile 목록에 저장
- library에서 base/surface/profile 클릭 시
  - 현재 popup draft에 복사

## Notes
- 현재는 UI mockup 단계이며 assignment는 브라우저 메모리 상태로만 유지된다.
- BSDF 등록도 현재는 파일 업로드 저장이 아니라 파일명 기반 등록 mockup이다.
