# Web UI v0.3.2 ROI split view

## Summary
- Removed the extra `Load selected CAD` button.
- `Import CAD` now represents the actual load action for the user flow.
- Split the right-side viewer into:
  - full CAD view
  - ROI view

## UX intent
- Reduce control clutter in Step 1.
- Match a more CAD-like mental model:
  - import full model
  - inspect full model
  - define ROI
  - review ROI separately

## Behavior
- After CAD import, the model loads immediately.
- The full CAD view always shows the whole mesh.
- The ROI view only renders currently selected faces.
- Both views share the same rotate/zoom interaction.

## Notes
- This is still a lightweight canvas-based viewer, not a full CAD kernel viewer yet.
- ROI isolation is currently display-oriented highlight/isolation, not geometry editing.
