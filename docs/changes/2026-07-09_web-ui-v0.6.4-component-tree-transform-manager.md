# Web UI v0.6.4 Component Tree / Transform Manager

## 목적
- CAD 프로그램처럼 `선택(Component Tree)`과 `편집(Transform Manager)`을 분리한다.
- 여러 component에 각각 다른 move/tilt 값을 독립적으로 줄 수 있도록 UI 기반을 만든다.

## 반영 내용
- Web UI 버전을 `v0.6.4`로 갱신
- 좌측 패널을 탭 구조로 개편
  - `CAD`
  - `Components`
  - `Transform`
  - `ROI`
  - `Material`
- `Component Tree` 탭 추가
  - 3D viewer/component tree 연동 선택
  - `선택 해제`
  - `선택 부품 추가`
- `Transform Manager` 탭 추가
  - component별 독립 transform rule 목록 관리
  - active rule 편집
  - checked rule batch action
- 우선 반영 기능
  - `Copy from selected`
  - `Mirror X`
  - `Mirror Y`
  - `Reset selected`
  - `Group move`

## 우선순위 로드맵
1. Component Tree + Transform Manager 기반 rule 편집
2. Batch action 안정화
3. `Save scenario A/B`
4. `Before/After compare`
5. rule grouping / folder / assembly hierarchy 고도화

## 검증
- `py_compile run_web.py` 통과
