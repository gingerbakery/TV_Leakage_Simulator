# Web UI v0.3.3 ROI main view

## Summary
- Changed the viewer flow to prioritize ROI after selection.
- Full CAD view is the main panel immediately after import.
- Once the user defines ROI, ROI view becomes the main panel and full CAD view moves to a smaller inset window.

## Behavior
- `Import CAD` -> full model opens in the main viewer
- no automatic ROI preselection on load
- when ROI is selected:
  - ROI view expands to the main canvas
  - full CAD view becomes a mini window
  - ROI remains highlighted inside the mini full CAD view

## UX intent
- Match the real review flow more closely:
  - start from whole assembly
  - narrow down to ROI
  - then work mainly inside ROI while keeping assembly context visible
