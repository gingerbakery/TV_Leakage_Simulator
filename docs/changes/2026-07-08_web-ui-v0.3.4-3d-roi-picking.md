# Web UI v0.3.4 3D ROI picking

## Summary
- Added direct ROI face picking in the 3D viewer.
- Kept the existing left-panel ROI workflow.
- Added selection mode switching so the user can choose how ROI is controlled.
- Added a demo OBJ file and `Load demo CAD` button for quick testing.

## ROI selection modes
- `Combined (3D + left panel)`
- `3D click only`
- `Left panel only`

## Behavior
- Clicking a face in `Full CAD View` toggles that face into ROI.
- Clicking inside `ROI View` also toggles selected ROI faces.
- Left panel part checkboxes and manual face list remain available.
- Final ROI used for simulation is synced from the active selection mode.
