# Wireframe 면 투명도 조정 (Web UI v0.9.20)

## 변경 배경

- 개선된 Wireframe이 외곽선 판독에는 유리하지만 면이 완전히 불투명해 `Surface + Edge`와 시각적으로 유사했다.

## 변경 내용

- Wireframe 보조 면(`wirefill`)의 불투명도를 `100%`에서 `82%`로 낮췄다.
- CAD 경계선은 기존 밝기와 선명도를 유지해 형상 판독성을 확보했다.
- 깊이 테스트와 깊이 기록은 유지하여 겹친 면과 내부 삼각형 노이즈가 다시 나타나지 않도록 했다.

## 기대 효과

- Wireframe에서 내부 구조가 약하게 비치면서도 외곽선과 부품 형상을 쉽게 구분할 수 있다.
- Surface 및 Surface + Edge 렌더 모드와의 시각적 차이가 명확해진다.
