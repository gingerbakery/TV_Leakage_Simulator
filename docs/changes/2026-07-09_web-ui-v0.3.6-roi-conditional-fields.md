# Web UI v0.3.6 ROI conditional fields

## Summary
- Simplified the ROI selection area by showing only the inputs relevant to the current selection method.

## Behavior
- `3D view에서 선택`
  - hides component selection
  - hides face index direct input
- `Component 선택`
  - shows component selection
  - shows face index direct input
- `복합 선택`
  - shows both

## Intent
- Reduce clutter in the ROI target surface area.
- Keep the UI aligned with the user's chosen workflow instead of always showing every sub-option.
