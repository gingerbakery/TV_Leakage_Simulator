# Web UI v0.4.4 ROI 선택 분리 / surface 투명도 조정

## 목적
- `Surface`와 `Surface + Edge`의 차이를 더 명확히 만든다.
- 3D viewer 탐색과 ROI 선택이 섞여서 오작동처럼 느껴지는 문제를 분리한다.

## 반영 내용
- `Surface`
  - 면이 약간 투명하게 보이도록 조정
  - edge는 약하게만 표시
- `Surface + Edge`
  - 면은 더 진하게
  - edge는 더 선명하게
- ROI 선택 모드 기본값을 `Component 선택`으로 변경
- `3D view에서 선택` 모드일 때만 viewer 클릭이 실제 ROI 반영으로 동작
- 그 외 모드에서는 viewer 클릭이 ROI를 바꾸지 않고 하이라이트만 표시

## 기대 효과
- render mode 차이를 육안으로 더 쉽게 구분 가능
- 3D viewer를 자유롭게 돌려 보다가 의도치 않게 ROI가 바뀌는 문제 감소
