# Web UI v0.6.1 Enter 제출 차단 / Run 임시 OFF

## 목적
- CAD 툴 사용 습관처럼 입력창에서 `Enter`를 눌러도 페이지가 시뮬레이션 결과 화면으로 넘어가지 않도록 막는다.
- 당분간은 viewer/ROI/gap/move/tilt 조작에만 집중할 수 있도록 `Run simulation`을 임시로 비활성화한다.

## 반영 내용
- Web UI 버전을 `v0.6.1`로 갱신
- `Run simulation` 버튼을 `temporarily off` 상태로 비활성화
- form 내부에서 `Enter` 입력 시 submit을 막고, 현재 입력값만 반영하도록 수정
- Enter 입력 후 `updateGapSelectionStats()` + `drawViewer()`를 호출해 바로 화면에 반영
- Step 5 제목을 `Material`로 단순화하고 현재 Run이 꺼져 있다는 안내 문구 추가

## 기대 효과
- X/Y/Z move 또는 tilt 값을 넣고 `Enter`를 눌러도 결과 페이지로 이동하지 않음
- 사용자는 CAD 프로그램처럼 입력 → Enter → 화면 반영 흐름으로 계속 작업 가능
- UI 검토 단계에서 불필요한 시뮬레이션 실행을 방지
