# Web UI v0.6.8 - Transform Popup Merge

## Why
- `Components` 탭 안에 있던 `transform type`, `selection mode`는 실제 입력 흐름상 오른쪽 3D viewer transform 조작과 분리되어 있었다.
- 사용자 요청과 PPT 변경안에 맞춰, component row의 `Transform` 클릭 시 바로 3D viewer에서 작업이 이어지도록 정리해야 했다.

## What changed
- 왼쪽 `Components` 탭에서 아래 항목 제거
  - `transform type`
  - `selection mode`
- `Component Tree`에서 각 부품의 `Transform` 버튼 클릭 시
  - 해당 부품 하이라이트
  - `Transform popup` 오픈
  - popup 내부에서 아래 항목을 한 번에 설정 가능
    - `transform type`
    - `selection mode`
    - `X / Y / Z move`
    - `Rx / Ry / Rz tilt`
- 오른쪽 상단 `Transform preview` 패널은 입력창이 아닌 요약/안내 역할로 단순화

## Implementation notes
- `run_web.py`
  - `gapTargetMode`, `gapSelectionMethod` DOM 위치를 좌측에서 `cursorMovePopup`으로 이동
  - `viewerMovePanel`의 중복 move/tilt 입력 제거
  - popup 크기/레이아웃을 확장해 제어 항목 통합
  - `gapTargetMode` 변경 시 popup이 자동으로 닫히지 않도록 조정
